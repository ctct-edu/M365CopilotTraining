# Microsoft 365 Copilot ハンズオン研修

このフォルダーには、空間情報・測量分野の事業文脈に合わせた Microsoft 365 Copilot ハンズオンラボと演習用サンプルデータを格納しています。

この研修は[CTCテクノロジー](https://www.school.ctc-g.co.jp/)が主催しています。

## 研修の想定

- 対象者: 空間情報・測量分野の企業に所属する営業、技術営業、事業企画、プロジェクト推進、管理部門の担当者
- 到達目標: Outlook、Word、PowerPoint、Copilot Chat、Copilot Studio、Excel の基本操作と、業務で使えるプロンプト設計を体験する
- 共通シナリオ: サンプル自治体「自治体A」に対して、空間情報を活用した防災・インフラ管理DXを提案する
- 扱う業務要素: GIS、航空測量、衛星データ、3D都市モデル、道路・河川・橋梁管理、災害対応、庁内説明、提案書作成、案件管理

## ファイル構成

- `lab-manuals/`: 各演習の手順書が格納されています
- `sample-data/word-powerpoint/`: Word と PowerPoint の Copilot で使用するサンプルデータ
- `sample-data/excel/`: Excel の Copilot で分析する案件・活動のサンプルデータ
- `sample-data/outlook/`: Outlook 演習で使用するメール素材のサンプルデータ
- `sample-data/copilot-studio/`: Copilot Studio エージェントのナレッジソースとなるサンプルデータ

## 演習順序

説明用スライドの章立てに合わせ、演習は次の順序で実施します。Excel は補足演習として最後に実施します。

1. Outlook でフォローアップメールを作成する
2. Word で自治体向け提案書を作成する
3. PowerPoint で提案プレゼンテーションを作成する
4. Copilot Chat で自治体向け提案準備を行う
5. Copilot Studio で空間情報ナレッジ エージェントを作成する
6. Excel で案件・実証候補データを分析する

## 事前準備

この教材は、2026年7月時点で Microsoft 365 Copilot のライセンスを持つ組織アカウントで実施する前提です。

1. Microsoft 365 Copilot を利用できる組織アカウントでサインインできることを確認します。
2. Word、Excel、PowerPoint、Outlook で Copilot が表示されることを確認します。
3. `sample-data` 配下のファイルを OneDrive または SharePoint の研修用フォルダーにアップロードします。
4. 各ラボ手順書に記載されたファイル名が Copilot から参照できることを確認します。
5. 演習で使うファイルは、読み取り専用ではなく編集可能な状態で開いてください。
6. Outlook 演習は下書き作成までを対象とし、実在しない宛先や外部宛先には送信しません。
7. Copilot Studio 演習を実施する場合は、受講者または講師がエージェントを作成できる環境と権限を持っていることを確認します。作成権限がない場合は、講師デモまたは画面確認演習として実施します。
8. Copilot の UI 表記はテナントや更新状況により変わることがあります。画面上の表記が異なる場合は、同等の機能を選択してください。

## シナリオについて

本教材は、空間情報・測量分野で想定される「空間情報」「測量・計測技術」「ソフトウェア・アプリケーション」および、防災・減災、国土管理、インフラ管理、環境保全、DX・分析などの事業領域を参考に、研修用のサンプル案件として構成しています。実在する顧客名、企業名、人物名は使用していません。
