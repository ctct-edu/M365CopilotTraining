# 演習 2 - Copilot in Word で自治体向け提案書を作成する

> 所要時間: 30 分

---

## シナリオ

自治体Aに対して、防災・インフラ管理DXの提案書を作成します。演習で使用する案件情報とヒアリング内容をもとに、Word の Copilot を使って提案書の下書きを作成し、自治体の意思決定者に伝わる内容へ改善します。

## 使用するファイル

- `自治体A_案件概要.docx`
- `空間情報DX提案パッケージ_概要.docx`
- `ヒアリングメモ_自治体A.docx`

## この演習で学ぶこと

- Word の Copilot で参照資料から文書を作る方法
- 読み手、目的、章立てを指定した文書生成
- 導入ステップ、効果、リスクを表形式で整理する方法
- 自治体向けの丁寧で根拠ある文体に改善する方法

---

## 1. 新しい Word 文書を作成する

1. Microsoft Word を開き、OneDrive または SharePoint 上に **白紙の文書**を作成します。
2. 画面上部または本文内の **Copilot を使用して下書き** の入力欄を確認します。
3. 参照資料として、使用する 3 ファイルを追加します。
   - 参照資料を追加する操作が見当たらない場合は、Copilot の入力欄で `/` を入力してファイル名を検索するか、Copilot ペインからファイルを参照します。
   - ファイルが読み取り専用で開いている場合は、編集可能なコピーを作成してから進めます。

---

## 2. 提案書の初稿を作成する

```prompt
添付した資料に基づいて、自治体A向けの「防災・インフラ管理DX提案書」を作成してください。

読み手:
- 防災危機管理課
- 道路河川課
- 情報政策課
- 財政課

提案書に含める内容:
1. エグゼクティブサマリー
2. 現状課題
3. 空間情報活用による解決方針
4. 具体的なユースケース
5. 90日間の実証ステップ
6. 期待効果とKPI
7. 体制と役割分担
8. 留意点と次のアクション

自治体向けに、丁寧で分かりやすい文体にしてください。専門用語には短い補足を入れてください。
```

---

## 3. エグゼクティブサマリーを改善する

1. エグゼクティブサマリーの段落を選択します。
2. Copilot で編集を開き、次のプロンプトを入力します。

```prompt
このエグゼクティブサマリーを、部長級の意思決定者が1分で理解できるように、課題、提案、期待効果の順に書き換えてください。文量は現在の半分程度にしてください。
```

---

## 4. 実証ステップを表に変換する

```prompt
90日間の実証ステップを表形式にしてください。
列は「期間」「主な活動」「自治体A側の関与」「提案企業側の支援」「成果物」「完了条件」としてください。
```

---

## 5. KPI を具体化する

```prompt
期待効果とKPIのセクションを改善してください。
点検計画作成時間、現地確認回数、災害時の初動把握時間、台帳更新の鮮度、庁内説明資料作成時間の5つの観点でKPIを提案してください。
各KPIには測定方法と初期目標値を含めてください。
```

---

## 6. リスクと対策を追加する

```prompt
この提案書に「導入時の留意点と対策」セクションを追加してください。
留意点は、既存台帳データの品質、航空・衛星データの更新頻度、庁内権限、個人情報、災害時運用の5項目にしてください。
それぞれに具体的な対策を記載してください。
```

---

## 7. 最終要約を作成する

Copilot ペインを **チャットのみ**、**質問する**、または文書に直接変更を加えないモードに変更し、次のプロンプトを入力します。

```prompt
この文書を自治体Aの担当課へ送るメールの冒頭に貼れるよう、3つの箇条書きで要約してください。
```

---

## 実務で使えるプロンプトサンプル

```prompt
この文書を、部長級向けの簡潔な提案書に整えてください。冒頭に結論、期待効果、必要な判断事項を追加してください。
```

```prompt
選択した段落について、専門用語を減らし、非技術部門にも伝わる説明に書き換えてください。重要な用語には一文の補足を入れてください。
```

```prompt
この文書のリスク記述をレビューし、顧客に事前説明すべき注意点と、その対策案を表形式で追加してください。
```

## 完了

この演習では、Word の Copilot を使って、自治体向け提案書の下書き、表への変換、KPI具体化、要約を行いました。
