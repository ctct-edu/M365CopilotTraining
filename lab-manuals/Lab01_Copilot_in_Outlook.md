# 演習 1 - Copilot in Outlook で自治体向けフォローアップメールを作成する

> 所要時間: 25 分

---

## シナリオ

自治体Aとの初回ヒアリング後、顧客へフォローアップメールを送ります。また、社内の技術部門へ支援依頼を送り、次回説明に向けた準備を進めます。Outlook の Copilot を使って、メール下書き、トーン調整、コーチングを体験します。

## 使用するファイル

- `初回ヒアリングメモ_自治体A.md`
- `顧客フォローアップ素材_自治体A.md`

## この演習で学ぶこと

- Outlook の Copilot でメールを下書きする方法
- 自治体向けと社内向けでトーンを変える方法
- Copilot コーチングで文章を改善する方法
- 長いメモをメールに変換する方法

---

## 1. 顧客向けフォローアップメールを作成する

1. Outlook を開き、**新規メール**を作成します。
2. 宛先は空欄のままにするか、自分の研修用メールアドレスを入力します。この演習では送信しません。
3. 件名に `初回ヒアリングのお礼と次回ご確認事項` と入力します。
4. 本文に `初回ヒアリングメモ_自治体A.md` の内容を貼り付けます。
5. Copilot を開き、次のプロンプトを入力します。

```prompt
貼り付けたヒアリングメモをもとに、自治体Aの防災危機管理課へ送るフォローアップメールを下書きしてください。

条件:
- 自治体向けの丁寧な日本語
- 冒頭でヒアリングのお礼を述べる
- 合意事項を3点に整理する
- 次回までに確認したい事項を箇条書きにする
- 最後に次回打ち合わせ候補日を確認する
- 長すぎないようにする
```

---

## 2. トーンを調整する

```prompt
このメールを、より簡潔で読みやすい文面にしてください。
ただし、感謝と丁寧さは維持してください。
```

---

## 3. Copilot コーチングを使用する

1. 生成されたメール本文を選択します。
2. Copilot の **コーチング** または同等の改善提案機能を選択します。
3. トーン、明確性、読み手への配慮に関する提案を確認します。
4. 良い提案があれば適用します。

---

## 4. 社内技術部門への依頼メールを作成する

新しいメールを作成します。宛先は空欄のままにするか、自分の研修用メールアドレスを入力します。この演習では送信しません。

```prompt
次回の自治体A向け提案に向けて、社内の空間情報技術チームへ支援依頼メールを作成してください。

背景:
- 顧客は道路・橋梁・河川施設の点検効率化に関心がある
- 防災担当は豪雨時の状況把握と庁内共有を重視している
- 情報政策担当は既存GIS、台帳、個人情報の扱いを懸念している
- 次回説明では、3D都市モデルまたは衛星データを使った具体例を見せたい

依頼内容:
- 30分のデモ構成案
- 使用可能なサンプルデータの確認
- 個人情報と公開情報の切り分けポイント
- 次回打ち合わせへの同席可否

社内向けなので簡潔で、依頼事項が明確な文面にしてください。
```

---

## 5. 返信メールを作成する

`顧客フォローアップ素材_自治体A.md` に記載された顧客返信を本文に貼り付け、次のプロンプトを入力します。

```prompt
以下の顧客返信に対する返信メールを作成してください。
相手の懸念に共感しつつ、既存GISとの連携、台帳データ、個人情報の扱いについて次回打ち合わせで確認する旨を伝えてください。
必要な事前確認事項も3点だけ箇条書きにしてください。
```

---

## 6. 件名を改善する

```prompt
このメールの件名候補を5つ提案してください。
自治体の担当者が内容を把握しやすく、開封しやすい件名にしてください。
```

最後に、作成したメールは送信せず、下書きを破棄するか研修用フォルダーに保存します。

---

## 実務で使えるプロンプトサンプル

```prompt
以下の商談メモをもとに、顧客向けのフォローアップメールを作成してください。合意事項、確認事項、次回日程の3点が分かるようにしてください。
```

```prompt
このメールを、相手に追加作業を依頼する内容として丁寧に書き換えてください。依頼事項は箇条書きにし、期限と背景も入れてください。
```

```prompt
以下の長いメールスレッドを要約し、未対応事項、相手待ち事項、こちらが次に行うべきことに分けてください。
```

## 完了

この演習では、Outlook の Copilot を使って、顧客向けメール、社内依頼メール、返信メール、文章改善を行いました。
