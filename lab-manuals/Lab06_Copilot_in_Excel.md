# 演習 6 - Copilot in Excel で案件・実証候補データを分析する

> 所要時間: 30 分

---

## シナリオ

あなたは空間情報・測量分野の企業に所属する営業・事業推進担当者です。上期の自治体・インフラ関連案件、活動履歴、実証候補データを確認し、自治体Aへの提案優先度や、他案件への横展開方針を検討します。

## 使用するファイル

- `空間情報DX案件管理_2026上期.xlsx`

## この演習で学ぶこと

- Excel の Copilot でデータセットを要約する方法
- 案件、活動履歴、実証候補、KPI、地域課題を横断して分析する方法
- 案件の傾向、リスク、優先案件を抽出する方法
- グラフや新しい列を作成する方法
- 分析結果を提案アクションに変換する方法

---

## 1. Excel ブックを開く

1. OneDrive または SharePoint にアップロードした `空間情報DX案件管理_2026上期.xlsx` を開きます。
2. Excel の **ホーム** タブから Copilot を開きます。
3. Copilot ペインを **チャットのみ**、**質問する**、またはブックに直接変更を加えないモードに変更します。
4. `案件一覧`、`活動履歴`、`実証候補`、`効果測定`、`地域課題` の各シートが表示されることを確認します。主要なデータ範囲は Excel テーブルとして設定済みです。

---

## 2. データ全体を要約する

```prompt
このブックのデータセットを要約してください。
シートごとの内容、主要な列、提案判断に使えそうな指標を説明してください。
```

---

## 3. 優先案件を抽出する

```prompt
案件金額、成約確度、予定提案日、次アクション日の観点から、今週優先して対応すべき案件を5件選んでください。
選定理由と推奨アクションも表で示してください。
```

---

## 4. 案件リスクを見つける

Copilot ペインを **編集を許可する**、**ブックを編集**、または変更を提案・適用できるモードに変更し、次のプロンプトを入力します。

```prompt
案件一覧に「リスク判定」列を追加してください。
条件は、次アクション日が過去、成約確度が40%未満、または最終接触日から14日以上経過している案件を「要対応」としてください。
それ以外は「通常」としてください。
```

列の追加が自動で適用されない場合は、Copilot に「この条件を満たす Excel 数式を作成してください」と依頼し、生成された数式を新しい列に貼り付けます。

---

## 5. 月別パイプラインを可視化する

```prompt
予定提案月ごとの案件金額合計を集計し、縦棒グラフで表示してください。
提案テーマ別に比較できるようにしてください。
```

---

## 6. 自治体A案件を深掘りする

```prompt
自治体Aの案件について、関連する活動履歴と案件情報をもとに、次回ヒアリング前に確認すべき点を整理してください。
出力は「確認事項」「理由」「推奨する聞き方」の表にしてください。
```

---

## 7. 実証候補テーマを比較する

```prompt
実証候補シートをもとに、費用対効果、実現難易度、庁内調整のしやすさの観点から、優先すべき実証テーマを3つ提案してください。
```

---

## 8. KPI と地域課題を分析する

```prompt
効果測定シートと地域課題シートをもとに、実証で効果を示しやすいテーマを3つ選んでください。
各テーマについて、根拠となるKPI、対象地域、想定される説明ポイントを表で示してください。
```

---

## 9. 提案会議向けの要約を作る

```prompt
このブックから分かる提案会議向けの主要インサイトを5点で要約してください。
各インサイトには、根拠となるデータと推奨アクションを含めてください。
```

---

## 実務で使えるプロンプトサンプル

```prompt
案件一覧と活動履歴をもとに、次アクションが遅れている案件を抽出し、優先度順に並べてください。理由と推奨アクションも示してください。
```

```prompt
地域別、提案テーマ別、担当者別に案件金額と成約確度を比較し、今月注力すべき領域を提案してください。
```

```prompt
効果測定シートをもとに、顧客説明で使いやすいKPIを5つ選び、現状値、目標値、説明に使う一言メッセージを表にしてください。
```

## 完了

この演習では、Excel の Copilot を使って案件分析、リスク判定、可視化、提案アクションの整理を行いました。
