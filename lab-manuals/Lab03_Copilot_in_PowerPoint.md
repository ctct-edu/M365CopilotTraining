# 演習 3 - Copilot in PowerPoint で提案プレゼンテーションを作成する

> 所要時間: 30 分

---

## シナリオ

自治体Aへの初回提案に向けて、15分で説明できるプレゼンテーションを作成します。PowerPoint の Copilot を使い、Word の提案素材からスライドを生成し、自治体の関係者に合わせて構成と表現を調整します。

## 使用するファイル

- `空間情報DX提案パッケージ_概要.docx`
- `自治体A_案件概要.docx`
- Word 演習で作成した提案書

## この演習で学ぶこと

- Word 文書をもとに PowerPoint を作成する方法
- スライド構成の改善
- 技術的価値と行政効果を分けて伝える方法
- スピーカーノートと想定質問の生成

---

## 1. 新しいプレゼンテーションを作成する

1. PowerPoint を開き、OneDrive または SharePoint 上に **新しいプレゼンテーション**を作成します。
2. **ホーム**タブの Copilot アイコンをクリックします。
3. Copilot ペインで **+** から提案素材の Word 文書を追加します。
   - **+** が表示されない場合は、Copilot の入力欄で `/` を入力してファイル名を検索するか、**ファイルからプレゼンテーションを作成** に相当する操作を選びます。

---

## 2. 初回提案用スライドを生成する

```prompt
添付した資料に基づいて、自治体A向けの防災・インフラ管理DX提案プレゼンテーションを作成してください。

条件:
- 対象は防災危機管理課、道路河川課、情報政策課、財政課
- 15分で説明できる構成
- スライド数は8枚程度
- 空間情報、GIS、航空測量、衛星データ、3D都市モデル、実証計画を含める
- 自治体向けの落ち着いたビジネス表現にする
```

---

## 3. スライド構成を改善する

Copilot ペインを **チャットのみ**、**質問する**、またはプレゼンテーションを直接変更しないモードに変更して、次のプロンプトを入力します。

```prompt
このプレゼンテーションの構成を、自治体の意思決定者が理解しやすい順番になっているかレビューしてください。
改善すべきスライド順と理由を提案してください。
```

必要であれば、次のように依頼します。

```prompt
提案された順番に合わせて、スライド構成を再整理してください。
```

---

## 4. 導入背景スライドを追加する

```prompt
自治体のインフラ管理と防災対応でよくある課題を示す導入背景スライドを1枚追加してください。
老朽化、職員不足、台帳更新、災害時の状況把握を含め、危機感をあおりすぎない表現にしてください。
```

---

## 5. 技術説明を分かりやすくする

技術説明スライドを選択し、次のプロンプトを入力します。

```prompt
このスライドの専門用語を、自治体の非技術部門にも伝わるように書き換えてください。
GIS、衛星データ、3D都市モデルには一文の補足説明を入れてください。
```

---

## 6. 実証計画スライドを追加する

```prompt
90日間の実証計画を1枚のスライドにしてください。
フェーズは「対象施設の選定」「データ整備」「可視化・分析」「現場検証」「効果測定」とし、各フェーズの成果物を短く示してください。
```

---

## 7. スピーカーノートを作成する

```prompt
各スライドに、発表者が話すべき要点をスピーカーノートとして追加してください。
1スライドあたり3点以内、話し言葉に近い自然な日本語にしてください。
```

---

## 8. 想定質問を作成する

```prompt
このプレゼンテーションを聞いた自治体Aの担当者から出そうな質問を10個作成し、回答例も添えてください。
質問は、費用対効果、既存台帳との連携、個人情報、データ更新頻度、災害時運用、実証範囲を含めてください。
```

---

## 実務で使えるプロンプトサンプル

```prompt
このプレゼンテーションを、10分説明用に再構成してください。重要度の低い内容は付録に回し、意思決定者向けの流れにしてください。
```

```prompt
このスライドのメッセージを、技術説明ではなく業務効果が伝わる表現に変更してください。箇条書きは3点以内にしてください。
```

```prompt
各スライドのタイトルだけを見て、提案ストーリーが自然につながっているかレビューしてください。改善案も示してください。
```

## 完了

この演習では、PowerPoint の Copilot を使って、空間情報DX提案資料の作成、構成レビュー、メッセージ調整、発表準備を行いました。
